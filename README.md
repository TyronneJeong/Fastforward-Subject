# 데일리 스크랩 주식 정보
종목 코드 입력을 통해 최근 5영업일 간의 최고, 최저, 개장, 종가, 거래량, 거래일자별 데이터를 수집하여 리턴 한다.

### 설치
#### Dockerfiles

    docker run

#### 프로젝트빌드

    gradlew jib

### 사용도구
- Spring Boot 2.0
- Gradle
- MyBatis framewwork
- Swagger
- JUnit 5.0
- postgreSQL
- JiB Plugin
